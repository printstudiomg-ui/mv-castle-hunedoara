# MV CASTLE Hunedoara

Site static pentru restaurantul MV CASTLE, pregătit pentru GitHub Pages.

## Fișiere principale

- `index.html` - pagina principală
- `css/style.css` - stiluri și responsive
- `js/script.js` - meniu interactiv, lightbox, animații
- `assets/` - logo, fotografii și meniul PDF

## Publicare pe GitHub Pages

1. Creează un repository nou pe GitHub.
2. Încarcă toate fișierele din acest folder.
3. Mergi la Settings > Pages.
4. Alege branch-ul `main` și folderul `/root`.
5. Salvează. Site-ul va fi disponibil după câteva minute.

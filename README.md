# MV CASTLE Hunedoara

Variantă FIX pentru GitHub Pages. Toate fișierele sunt în rădăcina repository-ului, ca să nu se piardă CSS-ul și imaginile la upload.

## Cum se publică
1. Șterge fișierele vechi din repository sau încarcă aceste fișiere peste ele.
2. Upload la TOATE fișierele din acest ZIP direct în root.
3. Settings → Pages → Deploy from branch → main → /(root) → Save.
4. Link final: https://printstudiomg-ui.github.io/mv-castle-hunedoara/
